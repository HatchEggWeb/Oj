马踏棋盘

Description

在5＊5的棋盘上，行坐标为X，列坐标为Y的位置放置一个国际象棋中的马，请求出，该棋子不重复的访问完每一个格子的方法数。(1<=X,Y<=5)

Input

一行：用空格隔开的两个整数，分别表示马所在的行坐标X，列坐标Y。

Output

一个整数，访问每一个格子一次且仅一次的方法数。

Sample Input

fsf

Sample Output

sdfd

Tips

sdfd

Edit One Problem

输入整数 单位MS

输入整数 单位KB
